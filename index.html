3. Difference between Local storage and Session storage

Ans: On session storage data is only saved until the browser tab closed. On the other hand Local storage can save data as long as wanted. Even if the browser closed or shut the machine the data will be there. So we can say Local storage saves data permanently and Session storage saves data is not permanent. Local storage capacity is larger than the Session storage.
